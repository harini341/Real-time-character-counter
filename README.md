# Real-Time Character Counter

## Project Description

This project demonstrates a simple **text area with a real-time character counter** built using **HTML, CSS, and JavaScript**. The counter updates automatically as the user types, displaying the number of characters used and the maximum limit allowed.

## Features

* Real-time character counting
* Displays typed characters and maximum limit (e.g., 150/200 characters)
* Prevents users from exceeding the character limit
* Displays a warning message when the limit is reached
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. The user types a message in the text area.
2. JavaScript listens to the **input event**.
3. The script counts the number of characters typed.
4. The counter updates automatically in real time.
5. When the maximum limit is reached, a warning message is displayed and further typing is restricted.

## Usage

1. Download or clone the project.
2. Open the `index.html` file in your browser.
3. Start typing in the text area to see the character counter update in real time.
